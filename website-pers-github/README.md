# Pers Independen Indonesia - Website Media Pers

Website media pers independen yang siap digunakan dengan GitHub Pages. Desain modern, responsif, dan mudah dikustomisasi.

## Fitur
- ✅ Desain modern dan responsif
- ✅ Breaking news ticker
- ✅ Kategori berita (Politik, Ekonomi, Teknologi, dll)
- ✅ Filter berita per kategori
- ✅ Newsletter signup
- ✅ Search overlay
- ✅ Dark mode ready
- ✅ SEO optimized
- ✅ Fast loading

## Cara Menggunakan

### 1. Fork Repository
Klik tombol "Fork" di pojok kanan atas halaman GitHub.

### 2. Rename Repository
Ganti nama repository menjadi `[username].github.io`
Contoh: `joko123.github.io`

### 3. Aktifkan GitHub Pages
- Buka tab Settings
- Klik Pages di menu kiri
- Pilih branch "main" atau "master"
- Klik Save

### 4. Tunggu 2-3 menit
Website akan live di `https://[username].github.io`
